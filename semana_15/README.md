# Semana 15 - Tarea: Aplicación GUI de Lista de Tareas

Esta carpeta contiene una aplicación de escritorio desarrollada con Tkinter para gestionar una lista de tareas.

## Archivo principal

- `tarea_lista_tareas.py`: script de la aplicación.

## Funcionalidades

- Añadir una tarea desde el campo de entrada.
- Presionar Enter para añadir la tarea rápidamente.
- Seleccionar una tarea y marcarla como completada.
- Seleccionar una tarea y eliminarla.
- Doble clic en una tarea para alternar su estado completado.

## Ejecutar la aplicación

1. Abrir una terminal en el directorio del repositorio.
2. Ejecutar:
   ```bash
   python semana_15/tarea_lista_tareas.py
   ```

## Notas

- El estado de las tareas completadas se refleja con un prefijo `✓` y color verde.
- Esta solución cumple los requisitos de interfaz gráfica y manejo de eventos.
