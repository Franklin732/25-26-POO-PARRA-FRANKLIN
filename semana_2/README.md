# Técnicas de Programación - Semana 2

Este repositorio contiene ejemplos prácticos de las cuatro técnicas fundamentales de la Programación Orientada a Objetos (POO):

- **Abstracción**: Uso de clases abstractas para definir interfaces comunes.
- **Encapsulación**: Protección de datos mediante atributos privados y métodos de acceso.
- **Herencia**: Clases derivadas que reutilizan y extienden funcionalidad.
- **Polimorfismo**: Misma interfaz, comportamientos diferentes según el objeto.
