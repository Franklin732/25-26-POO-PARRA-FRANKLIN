# Semana 13 - Tarea GUI Básica

Esta carpeta contiene la implementación de la tarea de **Semana 13**: desarrollo de una aplicación GUI básica usando Tkinter.

## Descripción
La aplicación permite:

- Ingresar texto en un campo.
- Agregar el texto a una lista mediante un botón "Agregar".
- Eliminar elementos seleccionados con "Limpiar selección".
- Limpiar toda la lista con "Limpiar todo".
- Guardar la lista a disco con "Guardar lista" y cargarla con "Cargar lista".

## Ejecución
1. Abre una terminal en la carpeta `semana_13`.
2. Ejecuta:

```bash
python tarea_gui_semana_13.py
```

> Requiere Python 3 instalado (Tkinter viene incluido en la mayoría de las instalaciones de Python en Windows).
