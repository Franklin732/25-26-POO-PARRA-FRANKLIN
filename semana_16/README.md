# Semana 16 - Tarea: Aplicación GUI para Gestión de Tareas con Atajos de Teclado

Esta entrega contiene una aplicación de escritorio desarrollada en Python con Tkinter para gestionar una lista de tareas.

## Archivos incluidos
- `tarea_gestion_tareas.py`: código fuente de la aplicación.

## Requisitos implementados
- Interfaz gráfica con Tkinter.
- Campo de entrada (`Entry`) para añadir tareas.
- Botones para añadir, completar y eliminar tareas.
- Lista (`Listbox`) que muestra tareas.
- Atajos de teclado:
  - `Enter` → Añadir tarea.
  - `C` / `c` → Marcar tarea como completada.
  - `D` / `d` o `Delete` → Eliminar tarea.
  - `Escape` → Cerrar aplicación.
- Feedback visual:
  - Tareas completadas muestran `✅` y se colorean en gris.

## Ejecución
1. Abrir terminal en `25-26-POO-PARRA-FRANKLIN`.
2. Ejecutar:
```bash
python semana_16\\tarea_gestion_tareas.py
```
3. Interactuar con la GUI.
